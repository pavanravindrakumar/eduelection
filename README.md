# VoteWise AI 🗳️

## Problem Statement
Election Process Education Assistant

VoteWise AI is an interactive voter education platform built to help users understand the election process, verify eligibility, locate polling booths, simulate voting, and combat misinformation through a simple and accessible interface.

Built for Hack2Skill Visual Prompt Wars.

---

## Key Features

### Core Features
- AI-powered Election Guidance Assistant  
- Voter Eligibility Checker  
- Interactive Election Timeline  
- Voting Simulator (EVM + VVPAT awareness)  
- Personalized “Your Journey” guided flow  

### Bonus Innovations
- Constituency / Polling Booth Finder  
- Voice-enabled assistant support  
- Election misinformation Fact Checker  
- WhatsApp-forward verification support
