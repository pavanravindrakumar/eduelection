# VoteWise AI 🗳️

## Problem Statement
Election Process Education Assistant

VoteWise AI is an interactive voter education platform designed to help users understand the election process, verify eligibility, locate polling booths, simulate voting, and combat misinformation.

Built for Hack2Skill Visual Prompt Wars.

---

## Key Features

### Core Features
- AI Election Assistant (Gemini-powered guidance)
- Voter Eligibility Checker
- Interactive Election Timeline
- Voting Simulator (EVM + VVPAT awareness)
- Personalized “Your Journey” guided flow

### Bonus Innovations
- Google Maps Constituency / Booth Finder
- Voice-enabled AI assistant
- Election misinformation fact-checker
- WhatsApp-forward verification support
